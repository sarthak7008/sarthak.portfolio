📌 About This Project
This is my personal portfolio website — designed to showcase my skills, projects, education, and contact information. Built with a cyber‑inspired dark theme, smooth interactions, and a strong focus on visual detail.

The portfolio highlights my journey as a Diploma in Computer Engineering student and my passion for programming, entrepreneurship, and digital innovation.

🎨 Features
🔹 Custom Cursor Experience
Smooth‑lag cursor ring with dot interaction.

Hover and click states with dynamic scaling and color shifts.

🔹 3D Tilt on Project Cards
Cards respond to mouse movement with subtle 3D perspective rotation.

Depth and parallax effect on hover.

🔹 Animated UI Elements
Glitch effect on the hero name.

Gradient‑animated name with flowing cyan tones.

Sweeping shimmer and border animations on the avatar image.

Skill bars that fill on load.

🔹 Fully Responsive
Adapts to mobile, tablet, and desktop screens.

Mobile‑optimized navigation (hamburger menu not included, but nav links hide on small screens).

🔹 Contact Form
Integrated with FormSubmit for email delivery.

Validates and sends messages directly to sarthak.deokar31@gmail.com.
